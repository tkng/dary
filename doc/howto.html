How to use dary?

Create double-array

To build a double-array index file, you use command 'darybuild'. 

$ darybuild keywords.txt keywords.dary

keywords.txt should contain one keyword per line, need not to be sorted.

Search from double-array

You use command 'darysearch' for common prefix search by input string.

$ darysearch keywords.dary
(Input something)

Extra usage

If you want to use dary from your programs, your program should link to libdary. See dary.h for functions you can use with dary. You may want to see sample in sample directory.
