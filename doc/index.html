Dary, a Double-ARraY library

What's dary

Dary is a library which implemented a data structure called 'double-array'. Double-array is a concrete data structure to substantialize TRIE with compact space usage. You can use dary as a fast dictionary library. If you want to know more about Double Array, see references.

Features

  • Dynamic, fast array construction. Dary's element insertion is fast, comparable to hash table. (You can construct a Double Array which includes over 100,000 keys within 1.0 sec.)
  • Very small library size. (only 12kB after stripped.)
  • Both prefix-search and common-prefix-search are available.

Download

Requirement

To build a library from source, you need a C99 compliant C compiler. If you want to use test code, you need to install glib-2.0 before compilation of dary.
